A search query is the word or phrase a person enters into a search engine. Search query data helps teams understand what people are looking for before they land on a website.

In SEO analytics, search queries connect user intent to impressions, clicks, click-through rate, average position, and landing page performance.

## Search query examples

Examples of search queries include:

- web analytics glossary
- privacy friendly analytics
- how to calculate bounce rate
- google analytics alternative
- what is conversion rate
- best open source analytics

Each query reveals intent. Some queries are informational. Others are commercial or product-focused.

## Search query vs keyword

A search query is what the user actually typed. A keyword is the term a marketer or SEO tool targets. They often overlap, but they are not always identical.

For example, the target keyword may be "web analytics," while search queries include many variations and questions.

## How to use search query data

Use search query data to improve titles, meta descriptions, content structure, internal links, and landing page intent match. Then use website analytics to see whether query-driven traffic converts.
