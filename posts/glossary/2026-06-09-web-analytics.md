Web analytics is the collection and analysis of data about website traffic, visitor behavior, acquisition sources, conversions, performance, and user journeys. It helps teams understand what happens on a website and how that activity supports business goals.

At its simplest, web analytics answers questions such as: How many people visited? Where did they come from? Which pages did they view? Did they convert? Did the website perform well?

## Core web analytics metrics

Common web analytics metrics include:

- Page views
- Unique visitors
- Sessions
- Bounce rate
- Time on page
- Average session duration
- Traffic channels
- Referrers
- Conversion rate
- Custom events
- Funnel completion
- Revenue

No single metric tells the full story. Traffic volume, behavior, conversion, and technical quality should be analyzed together.

## What web analytics is used for

Teams use web analytics to:

- Measure marketing campaigns
- Improve landing pages
- Understand content performance
- Track conversions
- Find funnel drop-offs
- Monitor page speed
- Detect JavaScript errors
- Evaluate SEO traffic
- Analyze product usage
- Report business growth

The best analytics setup turns raw events into decisions. It helps you know what to improve next.

## Privacy-first web analytics

Traditional analytics often relied on cookies, third-party scripts, and advertising identifiers. Privacy-first web analytics takes a different approach: collect less data, avoid invasive tracking, respect visitors, and still provide useful insights.
